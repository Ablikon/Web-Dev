About This Project

This application was created as part of the Lab 6 assignment. It demonstrates the use of Angular's routing module and HTTP client for interacting with a RESTful API.

Technologies Used

  • Angular: Frontend framework
  • Angular Router: For navigation between different views
  • HttpClient: For communication with the JSONPlaceholder API
  • RxJS: For handling asynchronous operations

Features

  • View all albums from JSONPlaceholder
  • View details of a specific album
  • Update album titles
  • Delete albums
  • View photos of a specific album
